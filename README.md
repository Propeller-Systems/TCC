# PROPELLER SYSTEMS GE - GERENCIAMENTO EMPRESARIAL - TCC

O projeto consiste em facilitar o processo de gestão do funcionário e organizar eventos, momentos e datas importantes para a empresa.
Esse projeto está sendo realizado por 5 pessoas, nas quais 2 trabalham no lado do desenvolvimento Web, 2 no desenvolvimento do app Mobile, e 1 responsável pelo relatório do projeto.
Nosso objetivo é tornar a gestão dos funcionários algo fácil, onde todas as principais funções estaram disponíveis na sua frente, com o máximo de simplicidade possível, tudo para não se tornar extremamente complicado e evitar que o usuário se perca.
